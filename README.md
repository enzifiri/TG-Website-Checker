# TG-Website-Checker
Websitedeki durumu bildiren telegram botu

Node JS NPM
```
sudo apt-get update
sudo apt-get upgrade
cd ~
curl -sL https://deb.nodesource.com/setup_16.x -o /tmp/nodesource_setup.sh
sudo bash /tmp/nodesource_setup.sh
sudo apt install nodejs
```

Puppeteer / TG Kurulum
```
npm install puppeteer
npm install node-telegram-bot-api
sudo apt-get install -y libxss1 libx11-xcb1 libxcomposite1 libxcursor1 libxdamage1 libxi6 libxtst6 libnss3 libcups2 libxrandr2 libasound2 libpangocairo-1.0-0 libatk1.0-0 libatk-bridge2.0-0 libgtk-3-0
npm install selenium-webdriver
sudo apt-get install -y libgbm1

```
